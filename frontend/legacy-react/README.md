This folder archives an older React/Vite prototype that is no longer part of the active QuizArena product.

The live web app and the Android build both use the original static frontend from the repo root.

Notes:
- The backend `POST /auth/google` route still exists and is tested.
- The Google sign-in button you saw was part of this archived React prototype, not the active static frontend.
- If this prototype is ever revived, it will need its own dependency setup and routing pass before it is production-ready again.
